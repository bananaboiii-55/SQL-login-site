# SQL Injection Practice Lab

A hands-on learning environment to understand **SQL injection vulnerabilities** and how to prevent them.

## 📋 Overview

This project contains two Flask applications:
- **Vulnerable Version** (`app_vulnerable.py`) - Intentionally vulnerable to SQL injection for learning
- **Secure Version** (`app_secure.py`) - Uses parameterized queries to prevent attacks

Compare both versions to understand the difference between unsafe and safe code.
